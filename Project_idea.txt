# **Project: Inventory Demand Prediction Dashboard**

**Goal:** Build a dashboard where users can **add items to inventory**, **view them**, and **predict demand** using a simple ML model. FastAPI handles the backend API, Streamlit handles the dashboard, and a predictive model forecasts future item demand.

---

## **1. Core Features**

### **FastAPI (Backend)**

* CRUD operations on inventory items:

  * `GET /inventory` → list items
  * `POST /inventory/create` → add an item
  * `PUT /inventory/update/{id}` → update item
  * `DELETE /inventory/delete` → delete item
* Endpoint to **predict demand**:

  * `POST /predict_demand` → accept item features and return predicted demand

### **Streamlit (Frontend)**

* Show **current inventory table**
* Form to **add/update/delete items**
* Form to **input item features** (like price, brand, category) to get a **demand prediction**
* Display **prediction result** visually (chart, number, or graph)

### **ML Model**

* Keep it simple, e.g., **linear regression** or **decision tree**
* Predict **daily demand** based on:

  * Price
  * Brand
  * Item type/category
* Train a small model on **dummy or generated data** (or real dataset if you can get one)

## **4. Example ML Model (backend/ml_model.py)**

```python
import pandas as pd
from sklearn.linear_model import LinearRegression
import numpy as np

# Dummy dataset
data = pd.DataFrame({
    "price": [5, 10, 7, 3, 12],
    "brand_quality": [7, 9, 6, 5, 8],
    "category": [1, 2, 1, 1, 2],  # encoded
    "demand": [50, 30, 40, 60, 25]
})

X = data[["price", "brand_quality", "category"]]
y = data["demand"]

model = LinearRegression()
model.fit(X, y)

def predict_demand(price, brand_quality, category):
    return model.predict(np.array([[price, brand_quality, category]]))[0]
```

## **5. Streamlit Form Example**

```python
import streamlit as st
import requests

st.title("Inventory Dashboard & Demand Predictor")

# Add new item
st.subheader("Add Item")
name = st.text_input("Item Name")
price = st.number_input("Price", min_value=0.0)
brand = st.text_input("Brand")

if st.button("Add Item"):
    response = requests.post("http://127.0.0.1:8000/inventory/create/1", json={
        "name": name,
        "price": price,
        "brand": brand
    })
    st.write(response.json())

# Predict demand
st.subheader("Predict Demand")
price_input = st.number_input("Price", min_value=0.0)
brand_quality = st.slider("Brand Quality", 1, 10)
category = st.selectbox("Category", [1, 2])

if st.button("Predict"):
    response = requests.post("http://127.0.0.1:8000/predict_demand", json={
        "price": price_input,
        "brand_quality": brand_quality,
        "category": category
    })
    st.write("Predicted Demand:", response.json()["predicted_demand"])
```
